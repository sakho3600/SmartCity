>实现架构上没有采用mvp，mvvm、也没有使用模块化，使用了安卓原生的mvc实现的，其实模块化不错的，解耦分离，模块分开，打算下以后使用下，使用模块化搭建。

###### 二 功能：
- 主页智慧中心搭建
- 新闻中心 轮播，及其列表
- 组图模块搭建
- 快看列表，快看视屏
- 设置界面自定义view通用封装
- 分享集成
- 引导页动画
- view抽取
- 等等
